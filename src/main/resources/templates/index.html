Calculator App

This is a web application built using Spring MVC and Thymeleaf
that adds two numbers provided as path variables in the URL and
displays the result in a user-friendly web page.

How to Use:
Add two numbers by appending /add/{num1}/{num2} to the URL,
where {num1} and {num2} are the numbers to add.
Example: https://champdoox-spring-render.onrender.com/add/5/7
will calculate and display the sum of 5 and 7.
